## はじめに

### はじめに

- 怒りに振り回されない生活を送るために
	- 怒りは人生を壊す唯一の感情
		- 怒りという感情自体が悪ではない。
		- 怒りは人間が生まれつき持っている感情。
		- 生きていくうえで必要不可欠なもの。
	- 重要なのは怒りに振り回されずに、コントロールすること。
		- その方法が**アンガーマネジメント**
- 「怒りなんてコントロールできるわけない」
- 「自分は生まれつき怒りっぽい性格だから、絶対直らない」
	- アンガーマネジメントを使えば誰でも怒りをコントロールできるようになる。
- アンガーマネジメントは「怒り」という感情を論理的にとらえ、実践的な対処法を提示するもの。
	- 精神論ではない。
	- 誰でも習得できる技術。
- 怒りをコントロールする方法を4つに分けて解説する。
	1. とっさの怒りを抑える方法
	2. 怒らないための習慣
	3. ムダに怒らない心の持ち方
	4. 怒りを上手に伝える方法

	- 怒らないこと＝アンガーマネジメント
		- 大きな間違い

## 第1章 とっさの怒りを切り抜ける7つの対症療法

### 「怒り」への対処はアレルギーと同じ
- 体質改善の前にまずは対症療法
	- 怒りとアレルギーは性質が似ている
		- 同じ状況に遭遇したときに、過敏に反応してしまう人と、まったく気にしない人がいる。
	- 対処法も似ている。
		- 体質改善が必要
			- 時間がかかるし、必ず効果が出るとは限らない。
		- 体質改善に取り組む前に薬など即効性のある対症療法が優先。
	- アンガーマネジメントにおける対症療法と体質改善
		- 対症療法：思わずカッときてしまったときの怒りを抑える。
		- 体質改善：ムダに怒らなくなる考え方を身につける。
- 第1章の内容
	- とりあえず「今」「この場」の怒りを収める7つの対症療法
	1. 「6秒ルール」で怒りのピークをやりすごす
	2. 「グラウンディング」で関係ないことを考える
	3. 自分の動作を実行して冷静になる
	4. 客観的に怒りを「10段階」で評価する
	5. 「タイムアウト」で立て直しの時間をつくる
	6. 「自分との会話」で頭の中を切り替える
	7. 真っ白の紙を思い浮かべて「思考停止」する

### 1. 怒りのピークは6秒で終わる

- 「１，２，３…」とゆっくり6秒数えれば怒りはやり過ごせる

- 反射的な言動をやめるトレーニングをしよう
	- イラっとしたとき、もっともマズいのが、反射的に怒りを表現してしまうこと。
		- 「おい！お前」
		- 売り言葉に買い言葉
	- 反射的な言動は自分も相手もヒートアップさせてしまう。
		- さらに余計なことを言ってしまう可能性もある。
		- その結果、取り返しのつかない事態、最悪の結果をもたらしてしまう。
	- イラっとしたときに何より大切なこと
		- 「反射的な原動をしない」
	- 怒りの感情のピークは6秒と言われている。
		- イラっとくることがあったら6秒間だけ待ってみる。
		- 深呼吸する、心を無にして思考を止めるなどの方法もある。

### 2. 「今、この場所」に意識をくぎ付けにする

- 目の前のものをじっくり観察すると、怒りから目をそらすことができます

- グラウンディングで怒りから目をそらす
	- 私たちの意識は、時間や場所を超えてどこにでも行くことができる。
		- 急に思い出し怒りを感じることがあるのも、このため。
		- 逆に、怒りとは関係のないところに意識を向けることで、怒りから目をそらすことができる。
		- この手法を **「グラウンディング」** という。
			- 「今、この場所のことだけを考える」手法
	- 一番良いのは瞑想。
		- 瞑想は誰にでもすぐにできるものではない。
		- 訓練が必要。
	- おすすめは **「目の前にあるものをひたすら観察する」** 方法
		- 見慣れたものでも、観察してみると意外と考えるネタがある。
	- 目の前にあるものを利用して、自分の意識をその場に留め置く→グラウンディング
- 未来や過去ではなく「今」に意識を向ける
	- こう考えるとイライラはどんどん膨張していく。
		- 渋滞に巻き込まれたとき「一体いつ動き出すのか」
		- 誰かに何かを言われたことを思い出して「あのときこう言い返してやればよかった」
	- 未来のことではなく、過去のことでもなく、今、この場所のことだけを考えると、怒りの膨張を防ぐことができる。
		- スマホの情報を見たりするより、目の前の無機質な物、怒りとは関係のないことに意識を向けるのがおすすめ。

### 3. 自分の動作を実況する

- 頭の中で自分の動作を実況することで冷静になることができる
- グラウンディングの方法のひとつ。
	- 「自分自身の行動を実況中継する」
	- 自分の置かれている状況を実況することに集中する。
	- 実況に集中することで怒りのピークをやり過ごすことができる。
- 自分自身の解説役に
	- 自分自身を実況することで、自分の置かれている立場を客観的に見ることができる。

### 4. 怒りのレベルを評価する

- 怒りのレベルは10段階で数値化が可能です
- 尺度がないと判断できない。
	- 私たちは何らかの尺度、数値をもとに、物事をコントロールしている。
		- 体温や血圧が高いと薬を飲む。
		- 気温で着る服を決める。
		- 商品の価値を値段で見る。
		- 今月の来店客数や売上高を見て、次月の対策を立てる。
	- 私たちは思っている以上に多くの尺度に囲まれて生活している。
		- 尺度があると、対策を立てられるから。
			- この数字は大変だ。大丈夫だ。
			- この数字だから、こうしよう。
	- 誰もが当たり前のように体調をコントロール仕様とする。
		- 怒りを自分でコントロールしようとする人はあまりいない。
		- 「怒りの尺度がない」ことにも原因がある。
		- 自分の怒りの程度がわからないから判断できない。
- 怒りのレベルで自分を知る
	- イライラを感じたら自分の怒りを数値化してみよう。
		- その怒りが10段階でどのレベルなのかを記録する。
		- 尺度の目安のうち、どれに当てはまるかは自分の感性で決めてかまわない。
	- まずは練習
		- これまで怒った経験にレベルをつけてみる。
		- 「イライラした」「ムカついた」など、似たような言葉で表される過去の怒りでも、レベルに差があることに気が付くはず。
	- 怒りのレベルを測るメリット
		1. 怒りを客観的に見られるため、冷静になれる。
		2. レベル別の対策が取りやすくなる。
		3. 気づいていなかった怒り、押し込めてしまっていた感情に気がつく。
		4. 自分の怒りの傾向を知ることができる（自分にとって大切なことがわかる）
	- 頭の中だけで完結せず、SNSやアプリなどで記録するのもよい。
- 怒りの尺度
	- 穏やか
		- ストレスやイライラがない状態
	- 不愉快
		- イラっとしたり、不愉快な気分を味わっている状態
	- 腹が立つ
		- 前面には出していないが、相当な怒りを感じている状態
	- 爆発寸前
		- 怒りが前面に出て、我を忘れそうになっている状態
	- 最大級
		- 震えが止まらないなど、憤怒、爆発に達する状態

### 5. タイムアウトで流れを変える

- 一旦その場から離れることで気持ちが落ち着く
	- アメリカのアンガーマネジメントで最初に学ぶのがラン（=逃げる）という戦略
		- その場にいてトラブルに巻き込まれるくらいなら、危機回避を優先して逃げるほうがよい。
	- 逃げることを卑怯な手段と感じる人もいるかもしれない。
		- 怒りをコントロールするために退却戦略は有効。
		- 「君子危うきに近寄らず」
	- スポーツの試合で流れが悪くなったとき、タイムアウトすることがある。
		- わずかな時間に少し言葉をかける程度でも、それが流れを変えるきっかけになる。
- タイムアウトで良い人間関係を築く
	- 相手がいる場所で怒りを感じたら、一言断って席を立つ。
		- 戻る時間をしっかり伝え、不信感を与えないようにしましょう。
		- 自分の都合で席を外すというスタンスが大切。
	- 夫婦間や社内で意見が対立しがちな場合は、スポーツのようにあらかじめタイムアウトのルールを作っておくのがよい。
	- そもそも交渉や議論は勝ち負けではない。
		- 良好な人間関係を保ちながら互いの意見をすり合わせるものだと認識しておくことが必要。
